Bank Account Reconciliation Optimization

In this module, we've optimized the account_bank_reconciliation module of NovaPoint Group. This module depends on account_cutoff_base, account_voucher, report_webkit. You need to uninstall the module account_bank_reconciliation before install our module to avoid function conflicts. Link demo user/pass admin/admin

Main Features:

We've optimized the package account_bank_reconciliation by adding the following features:

  • After user cancel/amend a posted Journal Entries, user can click button Refresh to get the Bank Reconciliation updated
  • Provide an validation to load only uncleared journal items.
  • Provide a filter to display only Bank/Cash/Cheque accounts on drop-down list.
  • With Bank Account in foreign currency, system only shows Start Balance in currency and End Balance in currency
  • Redesign the Detailed Report and Summary Report to have very neat and nice reports with clear information displayed.
  • Redesign the form view to make the function to be more simple
